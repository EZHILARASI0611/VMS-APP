# VMS APP

A professional Python web application built using the Flask framework. You can read the [Official Flask Documentation](https://palletsprojects.com) to learn more about how Flask works.

---

## 🚀 Setup Instructions

Follow these steps to set up and run the project on your local computer:

### 1. Install Dependencies
Before running the application, you need to install the required Python packages. Open your terminal in VS Code and run the following command:
```bash
pip install flask flask-login
```

### 2. Run the Application
Start the local development server by executing this command in your terminal:
```bash
python run.py
```

---

## 🌐 How to Open the Project in Your Browser

Once your terminal shows that the server is running, you can access the application locally. 

Click the direct link below or copy and paste it into your web browser's address bar:

👉 [http://127.0.0.1:8000](http://127.0.0.1:8000)

---

## 🛑 How to Stop the Server
To turn off the web application, go back to your VS Code terminal and press:
* **`Ctrl + C`**
